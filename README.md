# CS3612-ML-Project
Project of CS3612: Machine Learning, Spring 2023

## Setup
```sh
conda create -n ml-project python=3.11
activate ml-project
conda install pytorch torchvision torchaudio pytorch-cuda=11.7 -c pytorch -c nvidia
conda install matplotlib
conda install tqdm
```
